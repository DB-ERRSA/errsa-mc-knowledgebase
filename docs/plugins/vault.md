# Vault

## Purpose

**Category:** `Infrastructure`  

Vault acts as a **bridge layer** between plugins, allowing systems like economy, permissions, and chat to communicate consistently without direct dependencies.

---

## Dependencies

!!! info "Required for this plugin to function"
- **Server:** Paper  
- **Plugins:**
  - Economy plugin (e.g., EssentialsX Economy)
  - Permission plugin (e.g., LuckPerms)
- **External services (if any):**
  - None
